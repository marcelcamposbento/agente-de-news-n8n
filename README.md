# 📰 Agente de Notícias com n8n

Automação criada com o **n8n** para coletar notícias via RSS, processar com IA e enviar diretamente por e-mail.

---

## 🚀 Funcionalidades

* 📡 Leitura automática de feeds RSS
* 🧠 Processamento e resumo de notícias com IA
* 📧 Envio automático por e-mail
* ⏰ Execução programada (Schedule Trigger)

---

## ⚙️ Tecnologias utilizadas

* n8n (automação de workflows)
* Gmail API (envio de e-mails)
* RSS Feed
* JavaScript (processamento de dados)
* IA para geração de texto

---

## 🧠 Como funciona

1. O fluxo é iniciado automaticamente por um gatilho de tempo
2. O sistema coleta notícias de um feed RSS
3. Filtra e organiza os dados
4. Utiliza IA para resumir ou gerar conteúdo
5. Envia um e-mail com as principais notícias do dia

---

## 📥 Como usar este projeto

1. Baixe o arquivo `.json` deste repositório
2. Abra o n8n
3. Clique em **Import workflow**
4. Selecione o arquivo baixado
5. Configure suas credenciais (Gmail, APIs, etc.)
6. Ative o workflow

---

## ⚠️ Importante

* Não esqueça de configurar suas credenciais no n8n
* Este projeto não inclui dados sensíveis por segurança
* Pode ser necessário configurar a API do Gmail no Google Cloud
